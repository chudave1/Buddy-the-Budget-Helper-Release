TESTING INSTRUCTIONS — Buddy the Budget Helper

NO USER LOGIN REQUIRED
This app does not use usernames, passwords, Microsoft accounts, or any publisher-
hosted login. There is no test account to create. All data is stored locally on
the PC in a SQLite database. The app works fully offline.

IMPORTANT: "Accounts" in this app means bank/credit card definitions (e.g.,
"Checking", "Citibank Visa") for tracking spending. It does NOT mean user
authentication and does NOT connect to any bank website.

FIRST LAUNCH
On first launch the main window is intentionally empty. This is expected — the
app creates a new local database automatically. No sign-in screen appears.

QUICK TEST (about 5 minutes) — load sample data:
1. Launch the app. No login is required.
2. Import sample categories:
   - Menu: Import → Categories
   - File: Sample_Database_Category_Import.csv (downloaded from GitHub above)
   - Mapping: use default / first available mapping
   - Expect: 34 rows imported successfully
3. Import sample history (budget vs. actual summary data):
   - Menu: Import → History
   - File: Sample_Database_History_Import.csv (downloaded from GitHub above)
   - Expect: 56 rows imported successfully
   - The main window should now show budget data for 2025–2026
4. View dashboards:
   - Menu: Analysis → Dashboards
   - Select "Actual vs Budget" or "Breakdown" — charts should display data
5. (Optional) Test transaction import and reconciliation:
   - Setup → Accounts → Add account (any name, e.g. "Citibank Visa")
   - Edit Import Mapping for that account (column mapping details and
     screenshots: see sample_database/README.md on GitHub)
   - Monthly Tasks → Import Transactions → Sample_Database_Transactions_Citibank.csv
   - Monthly Tasks → Review and Categorize Transactions

FEATURES TO VERIFY
- Main budget window (year/month grid)
- Category setup (Setup → Categories)
- Budget setup (Setup → Budgets)
- CSV import (Import menu)
- Transaction reconciliation (Monthly Tasks)
- Analysis dashboards (Analysis → Dashboards)
- Database backup/restore (File menu)
- Help opens documentation in the browser (only network use)
